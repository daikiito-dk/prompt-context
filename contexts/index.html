会話セッション用テンプレート（SESSION / SPEC / LEARNINGS）

チャットの冒頭に貼る運用向けの短い型です。上記プロダクト MD と組み合わせて使うと効果的です。

SESSION.md — セッション前提

プロジェクト概要、用語、今回のゴール、禁止事項を冒頭に固定して会話のドリフトを防ぎます。

テンプレートと使い方 →

SPEC.md — 仕様・受け入れ基準

入出力、エッジケース、完了の定義を箇条書き。実装依頼やレビュー依頼の共通文脈に使えます。

テンプレートと使い方 →

LEARNINGS.md — 蓄積ナレッジ

過去の失敗と回避策を短く記録。長いスレッドでも「いま効く教訓」だけを毎回注入できます。
