# 08_User_Persona_and_User_Journey_Deep_Dive

## 1. ターゲットペルソナ (Target Personas)
AIは、以下のユーザー属性に合わせた解説・コード生成のトーンを選択すること。

* **Persona A: フロントエンド・アプリケーション開発者**
    * **関心事:** React/Vue等のライフサイクルとの統合、型安全なAPI定義、UIへのローディング・エラー反映。
    * **AIへの要求:** `useEffect` 内でのクリーンアップ処理や、`TanStack Query` との組み合わせコード。
* **Persona B: Node.js バックエンド/ツール開発者**
    * **関心事:** サーバー間通信の信頼性、プロキシ設定、ストリーム処理、ファイルアップロード。
    * **AIへの要求:** `http.Agent` を用いたコネクションプール最適化や、`fs.createReadStream` を使った大容量通信。
* **Persona C: インフラ/SRE エンジニア**
    * **関心事:** タイムアウト、リトライ戦略、サーキットブレーカー、オブザーバビリティ（ログ出力）。
    * **AIへの要求:** インターセプターを用いた共通ロギングと、指数バックオフによる再試行ロジック。

## 2. ユーザージャーニー：標準的な実装フロー (Typical Implementation Journey)
AIは、ユーザーが現在どのフェーズにいるかを推論し、適切な助言を行うこと。

### フェーズ1：プロジェクト初期化
* **ユーザーの行動:** `axios.create` を行い、共通の `baseURL` と `timeout` を設定する。
* **AIの支援:** 「環境変数（.env）からの読み込み」と「インスタンスのモジュール化」を提案せよ。

### フェーズ2：認証・認可の組み込み
* **ユーザーの行動:** ログイン後のトークン（JWT等）を全リクエストに付与しようとする。
* **AIの支援:** 手動でヘッダーを渡すのではなく、「リクエストインターセプターによる自動付与」のコードを提示せよ。

### フェーズ3：高度なエラーハンドリング
* **ユーザーの行動:** 401（Unauthorized）時にトークンをリフレッシュし、元のリクエストを再試行したい。
* **AIの支援:** 「レスポンスインターセプターでの非同期エラー復旧」という高度なデザインパターンを解説せよ。

### フェーズ4：最適化とクリーンアップ
* **ユーザーの行動:** ページ遷移時の通信キャンセルや、パフォーマンス低下の解決を試みる。
* **AIの支援:** `AbortController` による一括キャンセルと、不要なデータ変換のスキップを提案せよ。

## 3. シナリオ別解決パターン (Scenario-Based Patterns)

| シナリオ | 発生する課題 | 推奨されるAxiosの解決策 |
| :--- | :--- | :--- |
| **一括ダウンロード** | メモリ消費の増大 | `responseType: 'stream'` または `blob` を使用。 |
| **不安定なネットワーク** | 通信の頻繁な失敗 | `axios-retry` を用いた自動再試行。 |
| **APIの同時実行** | 競合状態（Race Condition） | `AbortController` で古いリクエストを破棄。 |
| **マルチテナントアプリ** | 接続先URLの動的変更 | 複数の `Axios Instance` を目的別に作成・管理。 |

## 4. エッジケースの考慮事項 (Edge Cases)
AIは以下の状況において、ユーザーに注意を促すこと。
1. **タイムアウトの誤解:** ネットワーク層のタイムアウトと、サーバー処理待ちのタイムアウトを区別できているか。
2. **CORS制限:** ブラウザ環境での `withCredentials` 設定や、プリフライトリクエスト（OPTIONS）の影響。
3. **大容量JSONのパース:** `JSON.parse` がメインスレッドをブロックする可能性（Web Workerでの処理検討）。

## 5. AIへのインタラクション指示
1. **文脈の把握:** ユーザーが単発のリクエストコードを提示した際、「これは共通のインスタンス化を検討すべきフェーズではありませんか？」と一段上のレイヤーの提案を行え。
2. **ベストプラクティスの教育:** `08` に記載されたペルソナに応じ、そのユーザーが将来的に直面するであろう「フェーズ3以降」の課題をあらかじめ示唆せよ。
